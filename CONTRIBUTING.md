# Contributing to Sirsi Technologies

## Our Standard
Every Sirsi repository follows the **Sirsi Portfolio Standard v2.0.0**.

### Required Workflow
1. Run `/session-start`
2. Read Thoth memory (`.thoth/memory.yaml`)
3. Follow the 26 Universal Rules in §1 of GEMINI.md

### Code Quality
- All Go code must pass `golangci-lint`.
- All Node code must pass `eslint`.
- Minimum 80% test coverage for safety-critical modules.
